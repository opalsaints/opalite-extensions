# Chrome Web Store Listings

## Opalite for ChatGPT

**Short description (132 chars):**
Save AI-generated images from ChatGPT & DALL·E to your Opalite cloud library. Organize, upscale, and share from one dashboard.

**Detailed description:**
Opalite for ChatGPT automatically saves your AI-generated images to the cloud as you create them. No more losing creations in chat history.

Features:
- One-click save: Images are captured and stored as you generate them
- Cloud library: Access your images from any device via the Opalite dashboard
- Organize: Browse, search, and manage all your AI creations in one place
- Works with ChatGPT and DALL-E image generation
- Works with Sora video generation

Free tier includes 50 downloads/month. Upgrade for unlimited downloads and premium features like image upscaling and background removal.

Get started: Install the extension, sign in with Google, and open ChatGPT. Your images will be saved automatically.

Website: https://opalitestudios.com
Support: support@opalitestudios.com

## Chrome Web Store Submission Checklist

### Prerequisites
- [ ] Register Chrome Web Store developer account ($5 one-time fee)
- [ ] Prepare screenshots (1280x800) showing each extension in action
- [ ] Create promotional tile (440x280) with Opalite branding
- [ ] Prepare privacy policy URL: https://opalitestudios.com/privacy

### Per Extension
- [ ] Upload extension ZIP
- [ ] Fill in store listing (name, description, screenshots)
- [ ] Set category: "Productivity"
- [ ] Set privacy practices (data usage justification)
- [ ] Link privacy policy: https://opalitestudios.com/privacy
- [ ] Submit for review (1-3 business days)

### Privacy Practices Declaration
For Chrome Web Store privacy tab:
- **Storage**: Used for authentication tokens (JWT, refresh token)
- **Host permissions**: Required to detect and save images from the AI platform
- **Downloads**: Used to save images locally as backup
- **Notifications**: Used to alert when downloads complete
- **Identity**: Not actively used (can be removed if not needed)

### Post-Submission
- After approval, extensions auto-update via Chrome Web Store
- Remove `update_url` from manifest (already done)
- Monitor reviews and respond to user feedback
