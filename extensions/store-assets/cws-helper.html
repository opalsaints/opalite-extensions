Step-by-Step: Fill Out CWS Listing

Follow these steps for each extension (Gemini Suite, Grok Suite, ChatGPT Suite, DeviantArt Auto)

Open the Extension Draft

In the Developer Dashboard, click on the extension name (e.g., "Gemini Suite")

Upload the Package (zip)

Go to the "Package" tab → Click "Upload new package" → Select the zip file:
extensions/GeminiExt-opalite.zip, extensions/GrokExt-opalite.zip, or extensions/ChatGPTExt-opalite.zip

Fill in Store Listing

Go to the "Store listing" tab and fill in (in this order):
— Detailed Description (copy from this helper page)
— Set Category to Workflow & Planning
— Upload Store Icon (128×128)
— Add Promo Video YouTube URL (if available)
— Upload Screenshots (1280×800, up to 5)
— Upload Small Promo Tile (440×280)
— Upload Marquee Promo Tile (1400×560, if available)
— Set Homepage URL and Support URL
— Mature Content → No
→ Click "Save draft"

Fill in Privacy

Go to the "Privacy" tab and fill in (in this order):
— Single purpose description: paste from this helper page
— Permission justifications: paste each one (downloads, declarativeNetRequest, storage, notifications, scripting, activeTab, cookies, Host permission)
— Remote code: select "No, I am not using remote code"
— Data usage: check the 5 boxes listed in this helper (PII, Auth, Web history, User activity, Website content)
— Data usage certifications: check all 3 certification boxes
— Privacy policy URL: use per-extension URL (e.g., https://opalitestudios.com/docs/gemini/privacy)
→ Click "Save draft"

Fill in Distribution

Go to the "Distribution" tab:
— Visibility: Public
— Geographic distribution: All regions
— Pricing: Free (no in-app purchases)
→ Click "Save draft"

Fill in Test Instructions

Go to the "Test instructions" tab:
— Leave Username and Password blank (no login required for core features)
— Paste the Additional Instructions from this helper page
→ Click "Save draft"

Review & Submit

Check all tabs show green checkmarks. Once all tabs are green, click "Submit for review".
